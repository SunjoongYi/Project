1. List Button : USB 혹은 SD Card 의 파일 리스트 출력.
- Focus가 현재 재생중인 파일로 맞춰지고, 해당 폴더의 파일들을 출력한다.
- 리스트에서 선택되어진 동영상 파일을 재생
- 지원되지 않는 비디오 포맷이면 '지원되지 않는 비디오 포맷입니다.'라고 메시지 출력.


2. Play/Pause Button : 

Play 버튼 : 영상이 중지중 일 때 영상을 재생하며, Pause 이미지로 전환.
Pause 버튼 : 영상이 재생중 일 때 영상을 중지시키며, Play 이미지로 전환.


3. ProgressBar : 현재 재상 상태 표시. 특정 부분을 터치 함으로서 원하는 부분부터 영상 재생 가능. (5초에 한번씩 막대가 이동)


4. 재생 시간 : 현재 재생 시간과 전체 재생 시간 표시.


6. 앞, 뒤 Skip 버튼 : 현재 재생되고 있는 영상의 이전 부분, 혹은 다음 부분으로 10초를 건너띄는 기능 제공.
                   버튼을 띄지 않고 누르고 있을때는 연속적으로 해당 버튼의 기능을 한다.

7. 메뉴 페이드 인/아웃
메뉴 Fade out : 사용자가 스크린에 5초 이상 입력을 하지 않으면 화면상의 메뉴들이 페이드 아웃 된다.
메뉴 Fade in : 비디오 전체 화면 상태에서 사용자가 화면의 아무 부분이나 터치를 하면 사라졌던 메뉴들이 Fade in 된다.

8. 현재 재생중인 영상의 파일 이름 Display

디스플레이 세팅:

밝기 -

Auto : 센서값에 의해 주기적으로 자동으로 밝기를 변경한다.
Daylight : 오전 시간 (09:00 ~ 18:00) 에의 시스템에서 지정된 밝기
Night : 오후 시간 (18:00 ~ 09:00) 에의 시스템에서 지정된 밝기
Manual : 해당 항목이 선택되면 사용자가 지정한 밝기를 적용한다.

Font Size -
사용자가 지정한 폰트의 사이즈를 적용한다.

Reset -
디스플레이 설정을 시스템에 지정 되어 있는 디폴트 값으로 적용한다.


위젯 셋팅 :
사용자가 체크박스를 해제/체크 함으로써 런처 홈 화면에 나타나는 아이콘들을 정의한다.
